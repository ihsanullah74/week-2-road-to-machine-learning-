# Week 2 – Data Analysis, Cleaning & Visualization

This repository is part of my roadmap to becoming a Machine Learning Engineer.

##  Roadmap Goal
Learning Data Science and Machine Learning step by step.
Week 2 focuses on real-world data handling, cleaning, analysis, and visualization.

##  Dataset
**Food Habits Survey Dataset** – A survey of 125 college students covering eating habits, GPA, exercise, and lifestyle choices.

## 📁 Project Structure
Week2-Data-Analysis-Project/
├── food_coded.csv
├── analysis.ipynb
├── app.py
├── requirements.txt
└── README.md

##  What I Did
- Loaded and explored the dataset
- Cleaned messy data (GPA, weight columns had invalid entries)
- Handled missing values using median imputation
- Performed statistical analysis
- Created 5 visualizations (histogram, bar, boxplot, pie, heatmap)
- Built an interactive Streamlit dashboard

## 📈 Visualizations
- GPA Distribution (Histogram)
- Gender Distribution (Bar Chart)
- Exercise vs GPA (Boxplot)
- Calories Estimate (Pie Chart)
- Correlation Heatmap
